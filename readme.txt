Intarocryptocurrency Release Tree v1.2.0 (https://github.com/intarocryptocurrency/intarocryptocurrency)

Intarocryptocurrency is a PoW (NeoScrypt) / PoS / Master Node hybrid cryptocurrency, with a 5 million coin maximum supply, which is generated at a slower rate every 1 Million NTR, until it's limit is reached. Master Node cost 3000IC, block size is 20MB, and block time is ~2Minutes.

The block production is 50% PoW and 50% PoS, of which each PoS block is split between Masternode and Staker evenly.

Block Spacing: 120 Seconds

Maturity: 15 Blocks

Stake Minimum Age: 30 Minutes

Default Main-Net Network Ports:
Port: 44144
RPC Port: 44155
